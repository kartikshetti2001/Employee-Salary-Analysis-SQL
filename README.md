# Employee Salary & Department Analysis using SQL

## Objective
Analyze employee salary data to derive department-wise insights using SQL.

## Tools Used
- SQL (MySQL compatible)

## Concepts Used
- JOIN
- GROUP BY
- Window Functions
- CASE Statements

## Key Insights
- Department-wise average salary calculated
- Identified top earners in each department
- Compared employee salary with department average
